# Security Planning Skill

## 역할
보안 관점에서 설계를 검토하고 보안 요구사항을 정의한다.

## 검토 항목
1. 인증/인가 — 접근 제어 설계 적절성
2. 데이터 보호 — 민감 데이터 처리 방안
3. 입력 검증 — 외부 입력 처리 전략
4. 시크릿 관리 — 키/토큰 저장 방안
5. 통신 보안 — HTTPS, 암호화 전략

## security-pattern-library.md 참조
- CRITICAL/HIGH 패턴 사전 차단 설계
- 안전한 구현 패턴 강제
